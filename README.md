## Le projet

Wavely est un site web qui permet d'écouter des fréquences qui ont des effets bénéfiques et relaxants sur le corps et l'esprit.

## Technos

**Front-end**

- JavaScript
- React
- TailwindCSS
- React-player-audio
- Formik
- Yup
- React-toastify

**Back-end**

- Node
- Express
- Jasonwebtoken
- Nodemon
- Argon2
- Express-validator
- Dotenv
- Postman
- Multer
- Nodemailer
